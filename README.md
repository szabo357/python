# H1 Curso para aprender a programar con Python desde cero.

Python es un lenguaje de programación de alto nivel y de propósito general creado por [Guido Van Rossum](https://es.wikipedia.org/wiki/Guido_van_Rossum) a finales de los años 80s y fue liberado por primera vez el 20 de Febrero de 1991.
Actualmente es uno de los lenguajes de programación más populares y más utilizados en todo el mundo debido a que es fácil de aprender.

No importa si estas dando tus primeros pasos en la programacion o si eres un programador experimentado. Python es el lenguaje que debes aprender.
